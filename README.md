# Baldur's Gate EE Portrait Converter

Набор инструментов на Python для создания портретов персонажей для игры Baldur's Gate Enhanced Edition. 

## Возможности

✨ **Два режима работы:**

### 1. Автоматический конвертер (`bg_portrait_converter.py`)
- Автоматический поиск лиц на изображениях (OpenCV Haar Cascades)
- Интеллектуальное обрезание по лицу
- Пакетная обработка всех изображений в папке
- ⚠️ Может работать нестабильно на рисованных/стилизованных лицах

### 2. Ручной GUI редактор (`portrait_gui.py`) ⭐ Рекомендуется
- Графический интерфейс для выделения области портрета
- **Отдельные регулировки для Large и Medium размеров:**
  - **Large (L)**: полный портрет (210x330 пикселей)
  - **Medium (M)**: приближенное лицо (169x266 пикселей)
- Идеально для рисованных и стилизованных изображений
- Полный контроль над обрезкой

**Оба режима создают:**
- **Large**: 210x330 пикселей (BMP 24-bit)
- **Medium**: 169x266 пикселей (BMP 24-bit)
- Уникальные имена файлов с префиксом 'A' (A1234L.bmp, A1234M.bmp и т.д.)

## Использование

### Способ 1: Ручной GUI редактор (Рекомендуется) ⭐

Лучше всего для рисованных и стилизованных изображений:

```bash
# Интерактивный режим (будет запрошена папка)
python portrait_gui.py

# С указанием папки входа
python portrait_gui.py -i "C:\Path\To\Images"

# С указанием входной и выходной папок
python portrait_gui.py -i "C:\Path\To\Images" -o "C:\Output\Portraits"
```

**Как использовать GUI:**
1. Откроется окно с изображением
2. **Шаг 1**: Нарисуйте прямоугольник вокруг полного портрета для **Large** версии
3. **Шаг 2**: Нарисуйте прямоугольник вокруг приближенного лица для **Medium** версии
4. **Управление:**
   - ЛКМ (левая кнопка мыши): рисуем прямоугольник
   - ENTER: подтверждаем выделение
   - ESC: отменяем текущий шаг

### Способ 2: Автоматический конвертер

Для обычных фотографий реальных людей:

```bash
# Интерактивный режим
python bg_portrait_converter.py

# С указанием папки входа
python bg_portrait_converter.py --input "C:\Path\To\Images"

# С указанием входной и выходной папок
python bg_portrait_converter.py -i "C:\Path\To\Images" -o "C:\Output\Portraits"
```

**Опции командной строки:**
- `-i, --input` — Путь к папке с исходными изображениями
- `-o, --output` — Путь к папке для сохранения портретов (по умолчанию `input/bg_portraits`)

### Опция по умолчанию

Если просто нажать Enter без ввода пути, будет использована папка `input/` в папке проекта.

## Результаты

Портреты сохраняются в выходной папке с уникальными кодами (A + 4-значное число):

```
bg_portraits/
├── A1234L.bmp      (Large версия, полный портрет)
├── A1234M.bmp      (Medium версия, приближенное лицо)
├── A5678L.bmp
├── A5678M.bmp
└── ...
```

### Как используется в Baldur's Gate EE

Портреты с одинаковым кодом (например, `A1234L.bmp` и `A1234M.bmp`) будут отображаться как большой и средний размеры одного портрета в игре.

Скопируйте файлы в папку:
```
...Baldur's Gate Enhanced Edition\portraits\
```

Готовые портреты можно сразу использовать в игре! 🎮

## Поддерживаемые форматы входного изображения

JPG, JPEG, PNG, BMP, TIFF, TIF, WebP

## Особенности

- **Префикс 'A' с уникальным кодом** — Портреты получают имена типа A1234L.bmp для сортировки по алфавиту в игре
- **Отдельная обрезка для каждого размера** — В GUI режиме вы регулируете Large и Medium независимо
- **Заполнение без черных полос** — Изображение масштабируется и обрезается так, чтобы полностью заполнить портрет
- **Обработка ошибок** — Программа продолжает работу даже если возникают ошибки с отдельными файлами

## Решение проблем

**GUI не открывается или ошибка с окном**
- Убедитесь что установлен OpenCV: `pip install --upgrade opencv-python`

**Автоматический детектор не находит лица**
- Это нормально для рисованных/стилизованных изображений
- Используйте GUI редактор (`portrait_gui.py`) вместо этого
- Для автоматического детектора нужны реальные фотографии с ясно видными лицами

**Программа медленно работает**
- Обработка больших изображений требует вычислительных ресурсов, это нормально

## Технические детали

- **Детекция лиц** — OpenCV Haar Cascades (автоматический режим)
- **Обработка изображений** — OpenCV + PIL/Pillow
- **GUI** — OpenCV (cv2) для интерактивного выделения
- **CLI** — Click framework
- **Масштабирование** — LANCZOS для высокого качества
